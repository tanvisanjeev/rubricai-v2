About & Workflow
Step-by-step guide for researchers and educators.
How to Use RubricAI
Follow these steps in order for a complete evaluation run.
1
Data Setup (recommended)
Configure your course details, simulation context, evaluation purpose, and researcher expectations. This context is fed directly to the AI to improve scoring accuracy.
2
Rubric Framework
Upload your rubric file (.md, .txt, .pdf, .docx). All clusters and indicators are parsed automatically. Use checkboxes to select which indicators to evaluate — all selected by default.
3
Upload & Evaluate
Upload your transcript CSV. Columns are auto-detected. Click Edit Mapping if your column names are non-standard. Then click Run Evaluation.
4
Class Overview
Review scores for every participant across all evaluated indicators. Click any participant ID to open the full evaluation — scores, rationale, feedback, and transcript evidence. Download individual reports.
5
Summary & Charts
View cohort-level performance — KPI cards, indicator bar charts, score distribution, participant rankings, and AI-generated cohort summary. Export the full class report.
